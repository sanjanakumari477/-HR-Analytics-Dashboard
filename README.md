# 📊 HR Analytics Dashboard – Employee Performance & Attrition Analysis

## 🚀 Project Overview
This project was completed as **Task 2 of the Hunar Internship Program**.  
It focuses on building an **HR Analytics Dashboard** to analyze employee attrition and performance trends using data-driven insights.

The dashboard helps HR teams understand attrition patterns and make informed workforce decisions.

---

## 🎯 Objectives
- Analyze overall employee attrition
- Identify attrition trends by department, job role, and gender
- Support HR decision-making and retention strategies
- Present insights using an interactive dashboard

---

## 📌 Key Insights
- **Overall Attrition Rate:** 16%
- **Employees Left:** 238 out of 1,000
- **Highest Attrition Role:** Sales Representatives
- **Department with Highest Workforce & Attrition:** Research & Development
- **Gender Analysis:** Higher attrition among male employees
